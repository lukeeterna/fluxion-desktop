## NOTE REGIA GENERALI

**Palette colori:**
- Background dominante: `#0A0A0F` (nero profondo, quasi nero cosmico)
- Accent primario: `#6C63FF` (viola elettrico — brand Fluxion)
- Accent secondario: `#00E5A0` (verde acqua — successo, prenotazione confermata)
- Testo principale: `#FFFFFF`
- Testo enfasi: `#FFD700` (oro — prezzi, percentuali, numeri forti)
- Rosso pericolo (competitor): `#FF4444`

**Tipografia overlay:**
- Font: Helvetica Neue Bold / SF Pro Display Bold
- Titoli grandi: 72-96px, peso 900
- Sottotitoli: 42-52px, peso 700
- Label piccole: 28-32px, peso 500

**Audio:**
- Voce fuori campo: donna italiana, tono caldo ma autorevole, ritmo incalzante
- Musica: electronic ambient progressivo, parte bassa 0-5s, cresce a 20s, climax a 40s, risolve a 55s
- Sound design: suono notifica WhatsApp a 35s, suono cassa a 45s, chime positivo a 58s

**Montaggio:**
- Cut ritmo: ogni 2-4 secondi nelle scene action
- Transizioni: hard cut per contrasti (problema vs soluzione), dissolvenza per le scene calme
- Effetto testo: lettere che appaiono da sinistra a 60fps, niente fade lento
- Motion: camera virtuale leggero parallax sulle UI screenshot (Ken Burns modificato)

### SCENA 01 — IL PROBLEMA (0-5 secondi)

**Durata:** 5 secondi

**Descrizione visuale:**
Schermata nera totale. Appare il suono di un telefono che squilla (icona animata). Il telefono continua a squillare nel vuoto. Poi una singola linea di testo appare al centro, bianca su nero, come una sentenza.

**Voce fuori campo:**
> "Ogni volta che il tuo telefono squilla e non riesci a rispondere… perdi un cliente."

**Overlay testo:**
```
SQUILLO
NESSUNA RISPOSTA
CLIENTE PERSO
```
(Le tre righe appaiono in sequenza, una per uno, rapid fire — 0.4s ciascuna)

**Effetto:**
- Icona telefono vibrante, poi si spegne (diventa grigia, flatline)
- Sfondo rimane nero assoluto
- Suono: telefono che squilla 2 volte, poi silenzio totale — impatto psicologico

**3B — Calendario (26-31s):**

**3D — Offline (36-40s):**

## NOTE TECNICHE PRODUZIONE

### Audio
```
Voce: donna italiana, 30-45 anni, professionale, calore + autorevolezza
Microfono: cardioid condensatore (nessun riverbero)
Post-processing: EQ high-shelf 8kHz +2dB, compressore 4:1, de-esser
Musica: royalty-free (Artlist.io / Pixabay Music) tag: "corporate cinematic electronic"
Mix: voce -12dBFS, musica -24dBFS, sound fx -18dBFS
```

### Video
```
Risoluzione: 1280x720 (LemonSqueezy requirement) o 1920x1080 (landing page)
FPS: 25 o 30fps
Codec: H.264, bitrate 8-12 Mbps
Durata max: 75s
File size target: <50MB (ottimale per web)
```

### Localizzazione IT
```
Valuta: sempre "euro" o simbolo € (no USD)
Date format: GG/MM/AAAA
Telefono: formato italiano (+39 XXX XXXXXXX)
UI mockup: testo italiano in tutte le schermate demo
```

### Accessibilità
```
Sottotitoli: SRT file obbligatorio (LemonSqueezy lo richiede)
Contrasto: tutti i testi overlay rispettano WCAG AA (rapporto ≥4.5:1)
Font size minimo: 28px (leggibile su mobile small screen)
```
