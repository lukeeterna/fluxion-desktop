## Story spine (applicare a ogni pezzo visivo)

```
1. C'era una volta...      [il mondo/personaggio che lo spettatore riconosce]
2. Ogni giorno...          [lo status quo — cosa facevano]
3. Finché un giorno...     [il problema o trigger]
4. Per questo motivo...    [la conseguenza / il journey]
5. Finché alla fine...     [la risoluzione]
6. E da allora...          [il nuovo mondo / CTA]
```

## Formato script video

```
[00:00-00:03] VISIVO: [cosa si vede sullo schermo]
              AUDIO:  [narrazione o dialogo]
              NOTE:   [messaggio chiave / nota regia]

[00:03-00:08] VISIVO: ...
              AUDIO: ...
```

## Regole pacing

| Contesto | Velocità |
|----------|---------|
| Corporate demo | 1 concetto ogni 8-10 secondi |
| Social media | 1 concetto ogni 3-4 secondi |
| Transizioni | Mai dead air > 1 secondo |

Significato delle transizioni: cut = punto successivo, wipe = tempo che passa, dissolve = connessione.

## Selezione metafora visiva

Scegliere metafore che siano:
- Universali (non culturalmente specifiche)
- Immediatamente riconoscibili (zero spiegazione necessaria)
- Non abusate nella categoria

**Show don't tell:** prodotto che lavora > narratore che descrive il prodotto.
