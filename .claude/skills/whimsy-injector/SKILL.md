## Inventario delight (verificare prima di scrivere qualsiasi microcopy)

| Momento | Opportunità |
|---------|------------|
| Empty state | Insegnare E intrattenere. Mai "Nessun elemento trovato." |
| Error message | Riconoscere frustrazione + percorso avanti + umanità |
| Success state | Celebrare in modo proporzionale |
| Loading state | Rendere l'attesa più corta con qualcosa da leggere |
| Onboarding | Quando la personalità cementa la loyalty |

## Calibrazione whimsy

| Tono brand | Whimsy appropriato |
|-----------|-------------------|
| Professionale/serio | Wit asciutto, giochi di parole intelligenti solo |
| Friendly | Personalità, umorismo lieve, calore |
| Playful | Puns, personalità mascotte, Easter eggs |
| Mai appropriato | Umorismo su perdita dati, sicurezza, errori monetari |

## Formula microcopy

**Error:**
```
✓ [Riconosci] + [Cosa è successo] + [Cosa fare] + [opzionale: leggerezza]
  "Non è andato a buon fine. Il server ha avuto un momento. Riprova."
✗ "Errore 503: Service unavailable"
```

**Empty state:**
```
✓ [Contesto] + [Beneficio del riempirlo] + [Azione]
  "Ancora vuoto — i tuoi task futuri vivranno qui"
✗ "Nessun dato disponibile"
```

## Easter eggs che funzionano

- Sempre opzionali, mai intrusivi, sempre sorprendenti
- Konami code → qualcosa di inaspettato e brand-appropriato
- Variazioni stagionali sull'interfaccia
- Shortcut da tastiera con personalità
