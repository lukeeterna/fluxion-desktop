## Priorità piattaforma 2025-2026

| Formato | Funzione | Frequenza |
|---------|---------|----------|
| Reels | Reach primario | 3-4/settimana |
| Carousel | Max save rate | 2-3/settimana |
| Stories | Relazione quotidiana | Ogni giorno |
| Feed post | Brand identity | 1/settimana |

## Framework caption

```
Riga 1:    HOOK — stoppa lo scroll (stesse regole TikTok)
Righe 2-4: VALUE DELIVERY — il payoff promesso
Righe 5-6: CONNESSIONE — domanda o CTA
Post-copy: 3-5 hashtag niche (non 30)
```

**Cosa non fare mai:**
- "Metti like e seguimi per altri contenuti" (penalizzato)
- Quote motivazionali generiche senza angolo personale
- Hashtag stuffing
- Engagement bait

## Struttura carousel (max engagement)

```
Slide 1:    Claim audace o domanda (il hook)
Slide 2-8:  Un insight per slide, ognuna tease la prossima
Ultima:     Riassunto + CTA soft + "salva questo"
```
Terminare senza una ragione per salvare = carousel sprecato.

## Formula bio (150 char limit)

```
Riga 1: chi aiuti + come (specifico, non vago)
Riga 2: credibility signal (numero, risultato, riconoscimento)
Riga 3: CTA con link
```

## Output obbligatorio per ogni contenuto

Specificare sempre: formato | caption completa | hashtag (3-5) | CTA | note grafiche.
