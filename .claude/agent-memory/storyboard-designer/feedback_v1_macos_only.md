Il video V1 (1:51, `fluxion-tutorial-install-v1.mp4`) finisce con: "Per Windows trovi le istruzioni complete su FLUXION landing pages dev slash come installare." Questo è friction inaccettabile.

**Why:** Pubblico FLUXION è 80% Windows. Un tutorial che rimanda a una pagina esterna per la maggioranza degli utenti non funziona come strumento di onboarding post-acquisto.

**How to apply:** Ogni video tutorial installazione deve coprire ENTRAMBI gli OS in un unico flusso autocontenuto. Mai terminare con "per l'altro OS vai su...". Le sezioni OS devono essere chiaramente etichettate (titolo capitolo + badge overlay) per permettere seek diretto.
