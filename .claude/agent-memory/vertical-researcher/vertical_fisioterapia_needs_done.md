Research completed 2026-03-31. Output: `/Volumes/MontereyT7/FLUXION/.claude/cache/agents/vertical-needs-fisioterapista.md`

**Why:** FLUXION vertical `fisioterapia` under macro `salute` — needs clinical workflow support beyond simple booking.

**Key findings:**
- ~20.000-25.000 studi privati target, 73.000+ fisioterapisti iscritti Ordini
- No-show rate 23-30%, costo €150-250 per slot 45-60 min — reminder WhatsApp riduce fino a 70%
- Cicli di trattamento (10 sedute standard) non gestiti bene da nessun competitor
- STS (Sistema Tessera Sanitaria) obbligatorio ma ignorato — rischio sanzione
- Fattura elettronica VIETATA verso privati (D.Lgs. 81/2025)
- X-Fisiolab licenza perpetua €79 ma Windows-only, UI datata → FLUXION €497 ben posizionato
- SaaS competitor: FisioDesk €535/anno, OsteoEasy €378/anno, Appuntoo €420/anno
- Killer feature #1: ciclo trattamento con dashboard VAS trend + progress bar + stato pagamento
- Killer feature #2: nota clinica in <60 secondi con preset tecniche (Tecar, TENS, Laser)
- FLUXION offline-first = vantaggio vs tutti i SaaS (Sud Italia connettività instabile)
- Sara voice booking = nessun competitor ha questa funzione

**How to apply:** When designing fisioterapia scheda, prioritize: session note speed, cycle tracking dashboard, VAS progression chart, STS export, GDPR consent flow.
