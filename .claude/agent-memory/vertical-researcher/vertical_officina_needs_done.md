Deep-dive research written to `.claude/cache/agents/vertical-needs-officina.md` on 2026-03-31.

**Key findings:**

- 88.977 imprese attive nel settore autoriparazione (CNA Q1 2025), 96% microimprese (<10 dip.)
- Sotto-verticali: meccanico (~65k), carrozzeria (~22k), gommista (~12k), elettrauto (~8k)
- ~60% delle officine ancora su carta/Excel — enorme segmento "paper to digital"
- Solo 27,2% usa soluzioni IA/digitali nel lavoro quotidiano (CNA 2025)

**Top pain points:**
1. Telefono senza risposta mentre le mani sono sporche — killer feature: Sara voice agent
2. Storico veicolo disperso (carta, memoria, Post-it)
3. Scadenze revisione/assicurazione dimenticate — promemoria WhatsApp automatico
4. Chaos gommista nei picchi aprile/ottobre
5. No-show dopo aver ordinato ricambi
6. Notifica "auto pronta" ancora manuale
7. Fattura elettronica SDI obbligatoria dal 2019

**Campi scheda veicolo critici:** targa (lookup auto), km, scadenza revisione, scadenza assicurazione, tipo olio, codici DTC, storico interventi

**Nuovo obbligo 2025:** RENTRI — registro elettronico rifiuti (olio esausto, filtri, pneumatici)

**Competitor pricing:**
- Carta/Excel: €0 (35-60% del mercato)
- Licenza una tantum (X-Carservice, Progetto Autofficina): €59-69
- SaaS entry: €30-80/mese (FAST Officina, ClerkWeb, MTE Gest)
- SaaS completo: €80-200/mese (FAST Officina + banche dati, ClerkWeb, DECK)
- Enterprise: €200-500/mese (eSolver multi-sede)

**WTP:** Titolare odia abbonamenti mensili — lifetime €497 è percepito come 6-7 mesi di FAST Officina → valore altissimo

**FLUXION unique advantage:** Sara voice agent (unica nel segmento), WhatsApp 2-way nativo, lifetime license, setup in 10 minuti

**Why:** Foundation for Sara FAQ content for officina vertical, scheda veicolo field design, landing page copy for officina segment.

**How to apply:** When building officina scheda or Sara FAQ, prioritize: targa lookup, revisione scadenza tracker, WhatsApp "auto pronta", voice booking for tagliando/diagnosi. RENTRI is a differentiating feature if added.
