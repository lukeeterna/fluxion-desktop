  <action>
Create `scripts/generate-v6-srt.py` that generates SRT subtitles from the storyboard JSON and voiceover manifest.

**Logic:**
1. Read storyboard JSON and voiceover manifest
2. Calculate cumulative timestamps by summing scene durations from manifest
3. For each scene with voiceover text:
   - Split text into subtitle segments of max 80 characters per line, 2 lines max per subtitle
   - Calculate start time = cumulative duration of all prior scenes
   - Calculate end time = start time + this scene's voiceover duration
   - Distribute subtitle segments evenly across the scene duration
4. For dialogue scenes (scene_08):
   - Each dialogue line becomes a separate subtitle
   - Use timing from individual dialogue parts if available in manifest
5. Write SRT format:
   ```
   1
   00:00:08,500 --> 00:00:12,000
   Tu lo sai com'e'. Sei li' che lavori.
   Le mani occupate.

   2
   00:00:12,000 --> 00:00:16,500
   E il telefono squilla. Squilla ancora.
   ```

**SRT format rules:**
- Sequence number starting from 1
- Timestamp format: HH:MM:SS,mmm
- Max 2 lines per subtitle, max 42 chars per line (YouTube standard)
- Empty line between subtitles
- UTF-8 encoding with Italian accents preserved
