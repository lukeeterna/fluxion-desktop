2. Dati raccolti e finalità

2.1 Sito web (landing page)

Il sito web non utilizza cookie di profilazione, cookie di terze parti o strumenti di tracciamento. Non vengono raccolti dati personali durante la semplice navigazione del sito.

2.2 Acquisto tramite Stripe

Quando effettui un acquisto, i tuoi dati di pagamento (carta di credito, email) sono trattati direttamente da Stripe, Inc. in qualità di responsabile del trattamento dei pagamenti. FLUXION riceve da Stripe esclusivamente:

  • Indirizzo email fornito in fase di checkout
  • Conferma dell'avvenuto pagamento
  • Piano acquistato (Base o Pro)

Finalità: generazione della licenza software, invio dell'email di conferma con link di download e istruzioni di installazione.

Base giuridica: esecuzione del contratto (art. 6, par. 1, lett. b) GDPR).

2.3 Email post-acquisto

L'email di conferma viene inviata tramite Resend (servizio di invio email transazionali). L'indirizzo email viene utilizzato esclusivamente per:

  • Conferma ordine e invio licenza
  • Istruzioni di installazione
  • Eventuali comunicazioni tecniche di supporto

Non vengono inviate newsletter, email promozionali o comunicazioni commerciali non richieste.

2.4 Software desktop FLUXION

FLUXION è un software desktop installato localmente sul computer dell'utente. Tutti i dati dei clienti (anagrafiche, appuntamenti, fatture, schede) sono salvati esclusivamente nel database SQLite locale del computer dell'utente. Nessun dato dei clienti viene trasmesso a server esterni.

Le uniche comunicazioni di rete del software sono:

  • Verifica licenza (phone-home): all'avvio, il software verifica la validità della licenza tramite un server Cloudflare Workers. Viene trasmesso solo l'hash della licenza, non dati personali.
  • NLU (comprensione linguaggio): quando è attiva Sara (assistente vocale), le frasi pronunciate dall'utente vengono inviate a provider LLM (Groq/Cerebras) per l'elaborazione del linguaggio naturale. Queste richieste non contengono dati personali dei clienti finali.

2.5 Sara — Assistente vocale

Sara elabora le conversazioni vocali in tempo reale. Le registrazioni audio temporanee vengono eliminate immediatamente dopo l'elaborazione. Nessuna registrazione vocale viene conservata permanentemente.

3. Destinatari dei dati

I dati personali possono essere comunicati a:

  • Stripe, Inc. — elaborazione pagamenti (sede UE/US, adeguatezza ex art. 45 GDPR tramite Data Privacy Framework)
  • Cloudflare, Inc. — hosting sito web e verifica licenze (sede UE/US, clausole contrattuali standard)
  • Resend, Inc. — invio email transazionali (sede US, clausole contrattuali standard)

Non vengono venduti, ceduti o comunicati dati personali a terzi per finalità di marketing.

3-bis. Dati sanitari — Art. 9 GDPR

Per i verticali sanitari (odontoiatri, fisioterapisti, medici, psicologi, etc.), FLUXION consente la registrazione di categorie particolari di dati personali (dati relativi alla salute dei pazienti) ai sensi dell'Art. 9 del Regolamento UE 2016/679.

  • Base giuridica: consenso esplicito dell'interessato (art. 9, par. 2, lett. a) GDPR), raccolto dall'operatore sanitario durante la configurazione iniziale del software
  • Conservazione: i dati sanitari sono conservati esclusivamente sul dispositivo locale dell'operatore, sotto il suo esclusivo controllo. FLUXION non trasmette dati sanitari a server remoti
  • Responsabilità: l'operatore sanitario che utilizza FLUXION è Titolare autonomo del trattamento dei dati dei propri pazienti e deve garantire il rispetto della normativa vigente
  • Cancellazione: la funzione "Eliminazione GDPR" consente la cancellazione irreversibile di tutti i dati di un paziente, inclusi i dati sanitari

4. Conservazione dei dati

  • Dati di acquisto: conservati per il periodo previsto dalla normativa fiscale italiana (10 anni)
  • Email: conservata per la durata della licenza e per 12 mesi successivi
  • Dati locali nel software: conservati localmente dall'utente, sotto il suo esclusivo controllo
